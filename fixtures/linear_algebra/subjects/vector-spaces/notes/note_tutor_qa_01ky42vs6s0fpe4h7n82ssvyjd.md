# A basis of a space is a set of linearly independent vectors…

**Q (reader):** A basis of a space is a set of linearly independent vectors that span that space because every point (vector) that lies on this plane, or within this space, can be expressed as a linear combination of the basis vectors.

**A:** You’ve connected **spanning** to being able to express every vector as a linear combination. What do you think **linear independence** adds—why wouldn’t spanning alone be enough for a basis?
